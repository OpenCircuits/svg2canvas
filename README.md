# svg2canvas

[![npm][npm]][npm-url]
[![deps][deps]][deps-url]
[![Install Size][size]][size-url]
[![Downloads][downloads]][downloads-url]

svg2canvas is a utility library that helps to turn SVGs into fast run-time drawings for the JavaScript 2D canvas.

# Installation

```console
$ npm install svg2canvas --save
```
